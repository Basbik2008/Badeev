📌 О программе
Quadratic Equation Solver — это программа с графическим интерфейсом для решения квадратных уравнений вида ax² + bx + c = 0 🎯

⚙️ Что делает программа:
✅ Принимает коэффициенты a, b, c

🔢 Вычисляет дискриминант

🧠 Находит корни уравнения

📤 Выводит результат в понятном виде

🎨 Особенности:
🖥️ Простой и понятный интерфейс на русском языке

🛡️ Автоматическая проверка ввода данных

⚠️ Защита от ошибок (деление на ноль, ввод букв вместо чисел)

💻 Работает на Windows без установки Python

🚀 Быстрый расчёт за секунду

📖 Как пользоваться:
1️⃣ Введите коэффициенты a, b, c
2️⃣ Нажмите кнопку "РЕШИТЬ" 🟢
3️⃣ Программа покажет дискриминант и корни уравнения 📊

📊 Возможные результаты:
Случай	Результат
D > 0 🟢	Два корня → x₁ и x₂
D = 0 🟡	Один корень → x
D < 0 🔴	Корней нет (нет действительных корней)
